# 🍢 Sistema de Gestão - Espetinho 
> **Projeto de Extensão Universitária** focado na digitalização de microempreendedores locais.

Este sistema foi desenvolvido para automatizar o controle de estoque, faturamento e gestão de vendas do **Espetinho**, substituindo registros manuais por uma plataforma web moderna e segura.

---

## 🚀 Funcionalidades Principais
* **Controle de Vendas:** Registro detalhado de transações com data, hora e produto.
* **Gestão de Estoque:** Monitoramento em tempo real dos itens disponíveis.
* **Dashboard de Faturamento:** Visualização clara do faturamento total e metas.
* **Autenticação Segura:** Sistema de login para operadores e administradores.
* **Interface Premium:** Design responsivo focado na agilidade do atendimento.

## 🛠️ Tecnologias Utilizadas
O projeto utiliza o que há de mais moderno no ecossistema PHP e desenvolvimento Web:

* **Framework:** [Laravel 11](https://laravel.com)
* **Linguagem:** PHP 8.3
* **Frontend:** [Tailwind CSS](https://tailwindcss.com) (UI/UX)
* **Interatividade:** [Alpine.js](https://alpinejs.dev)
* **Banco de Dados:** MySQL
* **Segurança:** Laravel Breeze (Autenticação)

## 🎓 Contexto Acadêmico
Este software é o resultado prático de um **Trabalho de Extensão**. O objetivo foi aplicar conceitos de engenharia de software, banco de dados e design de interface para gerar valor real na comunidade local de Aparecida de Goiânia - GO.
